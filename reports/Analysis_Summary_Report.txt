                           Price per m² Predictor Identification

PROJECT OVERVIEW:
This analysis focused on identifying the best predictors for property price per 
square meter (PLN/m²) using machine learning models on a complete imputed dataset 
of Warsaw properties.

================================================================================
                              METHODOLOGY
================================================================================

STEP 1: DATA PREPARATION AND TARGET TRANSFORMATION
---------------------------------------------------
• Dataset: Complete imputed property data from Warsaw market
• Target Variable: Changed from absolute price to price per m² (PLN/m²)
• Rationale: 
  - Price/m² provides normalized comparison across different property sizes
  - Removes size bias that would dominate absolute price predictions
  - More meaningful for real estate valuation and comparison

• Feature Exclusions:
  - Surface area: Excluded as redundant when predicting price/m²
  - Original price: Excluded as it's used to calculate the target variable

STEP 2: DATA PREPROCESSING
---------------------------
• Categorical Variable Encoding:
  - Applied Label Encoding to string variables (e.g., 'material', 'district')
  - Example transformations:
    * 'cegła' (brick) → 0
    * 'beton' (concrete) → 1
    * Different districts → numeric codes
  
• Data Type Verification:
  - Ensured all features are numerical for ML algorithms
  - Verified no missing values remain after imputation

STEP 3: MACHINE LEARNING MODEL DEVELOPMENT
-------------------------------------------
• Train-Test Split: 80% training, 20% testing (stratified random sampling)
• Models Implemented:
  1. Random Forest Regressor
     - Parameters: 100 estimators, random_state=42
     - Tree-based ensemble method
  
  2. XGBoost Regressor  
     - Parameters: 100 estimators, max_depth=6, learning_rate=0.1
     - Gradient boosting algorithm

STEP 4: MODEL EVALUATION METRICS
---------------------------------
• R² Score: Coefficient of determination (proportion of variance explained)
• RMSE: Root Mean Square Error (average prediction error in PLN/m²)
• MAE: Mean Absolute Error (average absolute prediction error in PLN/m²)
