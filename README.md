# Warsaw Real Estate Market

End-to-end machine-learning pipeline for analysing and predicting residential property prices in Warsaw, Poland.

---

## Project overview

| Stage | Notebook | Description |
|-------|----------|-------------|
| 0 | `notebooks/00_data_ingestion.ipynb` | Scraping & raw data collection |
| 1 | `notebooks/01_sample.ipynb` | Data sampling & initial exploration |
| 2 | `notebooks/02_expore_missing_prices.ipynb` | Missingness patterns & MAR/MNAR analysis |
| 3 | `notebooks/03_modify.ipynb` | Cleaning, type-casting & feature engineering |
| 3a | `notebooks/03a_modify_data_imputation.ipynb` | Full imputation pipeline |
| 4 | `notebooks/04_model.ipynb` | Model training (Random Forest & XGBoost) |
| 5 | `notebooks/05_assess.ipynb` | Model evaluation & feature importance |

A Streamlit app (`app/estimate_app.py`) exposes the trained model as an interactive price estimator.

---

## Model & features

The model predicts **price per m² (PLN/m²)** using a Random Forest Regressor (~70% R²).

| Feature | Type | Description |
|---------|------|-------------|
| `district` | categorical | Warsaw district (label-encoded) |
| `building_age` | numerical | Age of the building in years |
| `dist_to_centrum_km` | numerical | Distance to city centre in km |
| `dist_to_metro_km` | numerical | Distance to nearest metro station in km |
| `rooms` | numerical | Number of rooms |
| `river_side` | binary | 0 = east bank, 1 = west bank of Vistula |

## Getting started

### 1. Clone & install

```bash
git clone https://github.com/klimanskanatalia/warsaw_real_estate_market.git
cd warsaw_real_estate_market

# Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install -r requirements.txt
```

### 2. Run the notebooks

Open the notebooks in order inside `notebooks/`.  
Each notebook reads from `data/` and writes outputs back to `data/` or `models/`.

### 3. Launch the price estimator

```bash
streamlit run app/estimate_app.py
```

Make sure `models/price_model.pkl` and `models/feature_columns.pkl` exist (generated by `notebooks/04_model.ipynb`).

## Using the Python API

```python
from warsaw_real_estate.data.loader import load_final_dataset
from warsaw_real_estate.models.predict import predict_price

df = load_final_dataset()

price_per_m2 = predict_price({
    "building_age": 20,
    "district": 3,            # e.g. Mokotów = 3
    "dist_to_centrum_km": 3.5,
    "dist_to_metro_km": 0.8,
    "rooms": 3,
    "river_side": 1,
})
print(f"Estimated price/m²: {price_per_m2:,.0f} PLN")
```
